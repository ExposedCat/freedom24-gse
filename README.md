# Freedom24 Panel Stocks

## Installation

### Manual

- Clone this repo

```bash
git clone git@github.com:ExposedCat/freedom24-gse.git && cd freedom24-gse
```

- Install dependencies

```bash
npm install
# Or
pnpm install
```

- Inject extension

```bash
./cli inject
```

- Log out & Log in

- (OPTIONAL) Update

```bash
git fetch && git pull && ./cli inject
```

- (OPTIONAL) Uninstall

```bash
./cli uninstall
```
